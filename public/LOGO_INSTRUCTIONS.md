# Azure Logo Setup Instructions

## How to Add Your Azure Tobacco Industrial FZCO Logo

1. **Save your logo file** as `azure-logo.png` in the `/public/` folder of this project
2. **Supported formats**: PNG, JPG, JPEG, SVG
3. **Recommended size**: 200x200 pixels or larger (square format works best)
4. **File name**: Must be exactly `azure-logo.png` (or update the src paths in the code)

## File Location
```
/public/azure-logo.png
```

## Where the logo appears:
- Header (top navigation bar)
- Sidebar (when expanded)
- Home screen (large prominent display)

## If logo doesn't appear:
- Check the file name is exactly `azure-logo.png`
- Check the file is in the `/public/` folder
- Refresh the browser
- Check browser console for any errors
